Dion Works Dokümantasyon

Kullanmaktan memnun olduğumuz ve sürekli geliştirdiğimiz Dion Temamızı, hazır WP kurulum paketi olan Dion WP Starter paketimizi, belirli standartlarımızı vede örnek kod parçalarımızın nasıl daha kolay ve verimleri kullanılabileceğini anlattığımız dokümantasyonlarımızdır.

Başlangıç

Sağlıklı ve belirli standartta doküman yazabilmemiz için ilk önce doküman yazmayı öğrenmemiz gerektiği için sıkça kullanacağımız doküman tanımlarını aşağıdan bakabilirsiniz.

İlgili tanımlamanın paragraf metinidir.

Örnek kod tanımlamaları;

// Örnek HTML kullanımı
<pre><code class="language-markup"></code></pre>

// Örnek CSS kullanımı
<pre><code class="language-css"></code></pre>

// Örnek JS kullanımı
<pre><code class="language-javascript"></code></pre>

// Örnek PHP kullanımı
<pre><code class="language-php"></code></pre>
